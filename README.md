
## Structure
- `data/` — raw and processed data (not tracked by git)
- `scripts/` — analysis scripts by track
- `results/` — pipeline outputs (not tracked by git)
- `docs/` — methods notes and decisions
- `envs/` — conda environment files

## Tracks
- Track A: RNAseq differential expression
- Track B: Comparative genomics and phylogenetics
- Track C: AlphaFold structural screening
